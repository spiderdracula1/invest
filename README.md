# Вариант №30. Инвестиции
## Требования
- Презентация должна быть выполнена с помощью Markdown, где это возможно
- Презентация должна быть выполнена с помощью фреймворка Reveal.JS
- Тема(оформление) обязательно должна быть вынесена в отдельный CSS файл
- Минимальное количество слайдов: 20
- Перед каждым разделом необходимо использовать слайд-разделитель
- Оформление презентации должно быть читаемым (например, черно-синий текст на черном фоне не допускаются)
- Обязательными являются титульный слайд, заключительный слайд
- Обязательно наличие меню для быстрого перехода к слайду, используя плагин reveal.js-menu. Меню должно быть рабочим. Все слайды должны иметь наименование в меню
- Обязательно наличие навигации: стрелки, номер слайда
- Обязательно наличие readme в корне презентации с заданием без указания баллов
- В случае сомнения самостоятельности выполнения работы - требуется обоснования спорных мест
- На слайде не должно быть скроллов
- Минимальное количество изображений в презентации - 3
- После создания репозитория - запрещается его удалять и перезаливать презентацию в новый репозиторий. Если есть проблемы - обращайтесь за помощью.
- Необходимо использовать локальную версию фреймворка Reveal.js
- Максимальная дата сдачи презентации 1 июня 2026 года.
## Система штрафов
- Перезаливка презентации в новый репозиторий после начала сдачи работы - 30 баллов
- Использование облачных библиотек и фреймворков - 30 баллов
- Сдача работы 2 июня и позднее - 30 баллов
